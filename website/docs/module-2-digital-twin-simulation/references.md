# References: Digital Twin Simulation Module

This document outlines the citation format and references used in Module 2: The Digital Twin (Gazebo & Unity).

## Citation Format

All references in this module follow the APA (American Psychological Association) citation style as specified in the project constitution.

### In-text Citations
- For direct quotes: (Author, Year, p. #)
- For paraphrasing: (Author, Year)
- For multiple authors: (Author1 et al., Year) for 3+ authors

### Reference List Format
- Alphabetical by first author's last name
- Hanging indent for multi-line entries
- Double-spaced throughout
